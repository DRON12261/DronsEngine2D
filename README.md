# DronsEngine2D

## В РАЗРАБОТКЕ НА НАЧАЛЬНОЙ СТАДИИ!

### DronsEngine2D - мой игровой движок для 2D игр, который я пишу при помощи C++&SFML.
По своей задумке это не должен быть убийца Godot, Unreal и т.д. по своей универсальности, удобству и функциональности. Я вижу этот проект как какой-то базовый фундамент движка (который я буду понимать "под капотом"), уже на основе которого я буду делать прототипы и сами игры. Т.е. в нем для этого уже будет базовая и оптимальная реализация игрового цикла, заготовки в архитектуре для мультплеера, многопоточности и поддержки модов, уже какой-то набор функционала для работы с физикой, графикой, какой-то математикой и т.д.
